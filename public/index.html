FoodEngine Will Find You Tastythings in Boston

FoodEngine is a dynamic web app for finding Boston-area restaurants and seeing/making reviews. FoodEngine was built from the ground up with Node.js, Express, MongoDB (with Mongoose), and AngularJS, and utilizing the Zomato developer API for location information on the client side. It's mobile responsive, features 4 user account types, persistent sessions, authentication with Passport.js, and you can login with Google+.

WebAppMaker Is Pretty Cool Too.

WebAppMaker is a blogger-clone single-page app allowing users to create simple blog pages with various types of widgets. WebAppMaker is also a MEAN stack application, and was the kernal of the coursework of Northeastern's CS 5610 Web App Development course. Do you love bootstrap? If so, you'll love it.
